# 하드코딩된 기능 및 한계점

* 링크의 미리보기는 노란색 박스로 처리되어 있습니다.
* 초기에 저장된 링크의 조회수나 주제 등의 메타데이터는 하드코딩되어 있습니다.
* 태그를 커스터마이징할 수 없습니다.
