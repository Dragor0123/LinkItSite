README
Link-it 애플리케이션은 사용자에게 편리한 링크 보관 및 공유 경험을 제공하는 것을 목적으로 합니다. 본 med-fi 프로토타입은 다음과 같은 기능을 제공합니다.

•	그리드 뷰, 리스트 뷰로 저장된 링크 확인
•	링크를 조회수 순, 주제 순으로 필터링 가능
•	조회수가 높은 링크는 상단에 큰 미리보기로 확인 가능
•	사용자가 직접 상단 고정한 링크도 상단에서 큰 미리 보기로 확인 가능

프로토타입은 다음과 같이 조작할 수 있습니다.
•	손가락을 사용한 탭, 스와이프로 링크 탐색
•	링크를 클릭하면 실제 링크로 이동
•	앱 최상단의 "KakaoTalk(Demo)" 버튼을 클릭하여 외부 메신저에서 Link-it!으로 링크를 불러오는 기능 테스트

본 프로토타입은 다음과 같은 한계를 가집니다.
•	실제 외부 링크 가져오기 불가능하고, KakaoTalk(Demo) 페이지를 통한 데모만 제공
•	링크 상단 고정 기능은 "최신순" 탭에 있는 Point-of-View 링크에 대해서만 체험적으로 가능

제작: 인간과컴퓨터상호작용 CHCI팀 이성재, 이희태, 김진우
